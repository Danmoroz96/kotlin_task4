# kotlin_task4
Kotlin Console Product Banner  This program simulates a product banner display using only **structured text output** in a Kotlin console application. It utilizes string formatting and padding to achieve the requested arrangement and layout.  

## Program Description

The application defines a `Product` data class and a `generateProductBanner` function. The text output is formatted to clearly display:

1.  A text placeholder simulating the **product image in the background**.
2.  The **name of the product** (centered).
3.  The **company name** (left-aligned at the bottom).
4.  The **contact info** (right-aligned at the bottom).
